# HolidayCheck

To run the app:

1) clone the repo
2) run 'npm install'
3) run 'npm start'

url : 'http://localhost:3003/hotels' will return the raw json data

url : 'http://localhost:3003/hotels/3' will return the hotels will stars = 3
